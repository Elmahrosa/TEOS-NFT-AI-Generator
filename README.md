# TEOS-NFT-AI-Generator

Official NFT generator for the TEOS Network Ecosystem by Elmahrosa International.

- Flask-based web interface

## Setup
1. Clone the repo
2. Create a virtual environment
3. Install dependencies
4. Run the app
